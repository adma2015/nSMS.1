NOTICE ======

MIDHORI nSMS.1 =====

nSMS.1 est une application Mobile autonome qui vous permet d'envoyer des messages SMS directement depuis votre mobile. Il vous permet de gérer votre entreprise de messagerie entière sans la nécessité d'utiliser un navigateur Web ou recharger votre compte d'un operateur téléphonique, parce que nSMS.1 contient toutes les fonctionnalités dont vous avez besoin pour exécuter vos propres campagnes de messagerie dans un environnement sécurisé, rapide et efficace, touche un public local et à l'international à un tarif réduit.
nSMS.1 peut être utilisé immédiatement après l'installation sans aucun processus de configuration! Il s'agit d'une méthode pour les SMS facile et rapide d'envoyer directement à partir d'un mobile.
Une fois qu'il est installé, il reste sur le mobile pour de bon, vous donnant une méthode pratique pour l'envoi SMS.

Fenêtre messaging ==================

+ Choisir entre deux options différentes pour l'importation de contacts:

Le statut de traitement -----------------
S'applique à la transformation au sein de nSMS.1 lui-même et la présentation du message au service SMS.
Liste de l'état de traitement / statut local) ----------------------------------------
SOUMIS: message envoyé au service
Accusés de réception (ou DLR) -------------------------
S'applique uniquement aux messages avec le «soumis» le statut, de sorte que la requête est envoyée pour l'ID de message applicable.
Lorsque l'état DLR final est reçu, l'envoi du message est considérée comme complète.

Liste des statuts DLR et les descriptions -------------------------------------
Message de NOT_SENT pas envoyé
Message envoyé ENVOYÉ
Message envoyé WAITING_DELIVERY, en attente du rapport de livraison
Message envoyé NOT_DELIVERED, pas livré
Message délivré envoyé et distribué
Réseau NOT_ALLOWED pas autorisé
En vrac ou de la passerelle NOT_ALLOWED déconnecté
INVALID_DESTINATION_ADDRESS adresse de destination non valide
Inconnue de livraison UNKNOWN_DELIVERY
Voie ROUTE_NOT_AVAILABLE pas disponible
INVALID_SOURCE_ADDRESS adresse source valide
NOT_ENOUGH_CREDITS pas suffisamment de crédits
En vrac NOT_ALLOWED des crédits de Message de NOT_DELIVERED rejetée
Message de NOT_DELIVERED expiré
Erreur système system_error émis par la passerelle
Format de INVALID_MESSAGE_FORMAT Message non valide
